1D Transfer Matrix Method Calculator

This website provides an interactive 1D transfer matrix method (TMM) calculator for calculating the transmittance spectrum, reflectance spectrum, and electric field distribution of a multilayer structure.
The default setting is for a 1D photonic-crystal cavity (or a Bragg cavity), which has been used to achieve the ultrastrong coupling regime in solid-state systems.
See Zhang, Qi, et al., Nature Physics 12.11 (2016): 1005-1011 and Li, Xinwei, et al., Nature Photonics 12.6 (2018): 324-329.

Layer structure

Enter the refractive-index list and thickness list in the same layer order.
Separate values using commas, spaces, or new lines. The two lists must have the same number of entries.
Use thickness in µm. Complex refractive indices are allowed, for example 3.42, 1, or 400+500i.
The default multilayer structure is Si / air / Si / air / defect layer / air / Si / air / Si, where n_Si = 3.42.

After changing the layer structure or frequency scan, click Run TMM to recalculate the spectra.

Transmittance and reflectance spectra

Peak selection

Use Peaks search freq min and Peaks search freq max to define the frequency range used for detecting cavity peaks from the calculated transmittance spectrum.
Change the peak number to view the electric field profile for the corresponding peak frequency.
