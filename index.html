Layer structure

Enter the refractive-index list and thickness list in the same layer order. Separate values using commas, spaces, or new lines. The two lists must have the same number of entries. Use thickness in µm. Complex refractive indices are allowed, for example 3.42, 1, or 400+500i.

After changing the layer structure or frequency scan, click Run TMM to recalculate the spectra.
